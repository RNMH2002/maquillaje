¿Que Es El Maquillaje?



El término maquillaje se utiliza para referirse a cualquier producto o cosmético utilizado en la práctica de retocar la piel de una persona, con el fin de embellecer, colorear, resaltar u ocultar ciertos rasgos faciales o corporales, en algunos casos. Hablar de maquillaje tiene que ver tanto con la práctica y técnicas de «pintar» la piel como un lienzo, como con cualquier elemento que se emplea para llevar a cabo este procedimiento. Así pues, decir que te vas a realizar un maquillaje (práctica) es tan válido como decir que necesitas tu maquillaje para hacerlo (cosméticos). Entre los elementos que componen el maquillaje destacan: los labiales, máscaras de pestañas, sombras para los ojos, lapices de labios, bases, correctores, polvos, colorete para las mejillas, delineadores… La mayoría de estos productos se realizan a partir de compuestos químicos, por tanto, es indispensable conseguirlos de fuente confiable.
